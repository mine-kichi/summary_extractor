# summary_extractor
## 概要
- Todoや質問事項など、日々の作業で出てきた内容の整理ツール

## 特徴
- 基本、テキストにつらつらと書き込んで行くのが好きな人用。何かをメモるときとかやっぱりテキストが軽いし早いから。。。
なので、これは、テキストに書いて、それを抽出して整理していくもの。
- テキストで書いたものを抽出してDBに保存、抜き出す機能がメインってことか。
- 完了、未完了をチェックすることもできそうだしなぁ。。。やっぱりDBにするか。。。
